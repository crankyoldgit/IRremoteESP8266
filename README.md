# IRremoteESP8266 v2.8.7 (Patched for ESP32 Arduino Core 3.x)

This is a fork of [IRremoteESP8266](https://github.com/crankyoldgit/IRremoteESP8266) with patches to support ESP32 Arduino Core v3.x.

### Summary of Changes

- Updated timer and interrupt handling for ESP32 Arduino Core 3.x compatibility.
- Version bumped to 2.8.7
- All original protocol support is preserved.

For full documentation, please refer to the [original repository](https://github.com/crankyoldgit/IRremoteESP8266).

---

Tested with:
- ESP32 Dev Module
- Arduino Core v3.3.0+
- Arduino IDE v1.8.19 and v2.3.6
